This is a modern and interactive social media landing page designed to showcase all my profiles in one place with a clean UI and smooth animations.

## ✨ Features
- 🌌 Animated particle background  
- 🎨 Glassmorphism design  
- ⚡ Fast & lightweight  
- 📱 Fully responsive  
- ⌨️ Typing animation  
- 🔗 Social media links
